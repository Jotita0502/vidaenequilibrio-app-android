# Vida en Equilibrio - Aplicación Android

## Descripción

Vida en Equilibrio es una aplicación móvil desarrollada en Android que permite a los usuarios gestionar recetas saludables, controlar productos de inventario, visualizar métricas y administrar información personal mediante una interfaz intuitiva.

## Funcionalidades

* Registro e inicio de sesión de usuarios.
* Gestión de recetas saludables.
* Administración de productos.
* Visualización de métricas y estadísticas.
* Perfil de usuario.
* Consumo de API REST.
* Persistencia de datos mediante MySQL.

## Tecnologías Utilizadas

* Java
* Android Studio
* Android SDK
* RecyclerView
* Fragments
* REST API
* PHP
* MySQL
* XAMPP
* SharedPreferences
